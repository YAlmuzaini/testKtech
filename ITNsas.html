Information Systems and Technology


Network System Maintenance and Security

General Description: The Network System Maintenance and Security (NSMS) major provides fundamental knowledge and application in preparing students for a career in technical and professional areas of network systems engineering. In addition to the program outcomes for the Diploma in Information Systems and Technology, NSMS graduates are expected to gain the major-specific outcomes:

  • NSMS-1. Create, design, install, implement, maintain and administer different systems in a network environment.
  • NSMS-2. Design, install, implement, maintain and manage network operating systems and server systems.
  • NSMS-3. Provision basic and advanced services for servers and network management.
  • NSMS-4. Analyze, implement, and administer network systems security using Microsoft systems.

Job opportunities: Operating System Technician - Technical Support Specialist - Technical Specialist
